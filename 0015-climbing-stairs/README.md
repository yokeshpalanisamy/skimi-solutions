# 15. Climbing Stairs

**Difficulty:** Medium

**Tags:** Dynamic Programming, Math

## Problem

You are climbing a staircase with `n` steps. Each time you can climb either 1 or 2 steps. Return the number of distinct ways to reach the top.

## Examples

**Example 1:**

```
Input: n = 2
Output: 2
```

1+1, or 2

**Example 2:**

```
Input: n = 5
Output: 8
```

## Constraints

- 1 ≤ n ≤ 45
