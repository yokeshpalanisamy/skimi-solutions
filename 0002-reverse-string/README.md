# 2. Reverse String

**Difficulty:** Easy

**Tags:** String, Two Pointers

## Problem

Write a function that reverses a string. The input string is given as an array of characters `s`.

## Examples

**Example 1:**

```
Input: s = ["h","e","l","l","o"]
Output: ["o","l","l","e","h"]
```

## Constraints

- 1 ≤ s.length ≤ 10⁵
